Website of the [react-ui-icheck][] library.

- [Production](#production)
- [Development](#development)

```bash
$ git clone git@github.com:victorpopkov/react-ui-icheck-website.git
$ cd react-ui-icheck-website/
$ yarn install
```

## Production

```bash
$ yarn build
$ yarn start
```

## Development

```bash
$ yarn dev
```
